# Aut_Sci_PPT 样式规范 (Style Specification)

本项目遵循经过实测高度满意的学术 PPT 视觉规范。

## 1. 页面基础参数
- **尺寸**：13.33 × 7.5 英寸（16:9 宽屏）
- **边距**：四周 0.5 英寸（用于文字 safe area）

## 2. 配色方案 (Color Palette)
| 用途 | 色值 | 说明 |
|------|------|------|
| **主题色 (Primary)** | `#1E3A5F` | 深蓝色，用于标题栏背景、导航栏及重点标签 |
| **高亮色 (Accent)** | `#EE0000` | 红色，仅用于正文核心关键词强调 |
| **正文色 (Text)** | `#000000` | 纯黑，用于普通列表项和描述文字 |
| **白体 (Reverse)** | `#FFFFFF` | 白色，用于导航栏文字或深色背景上的标题 |

## 3. 字体规范 (Typography)
| 位置 | 字体名称 | 样式要求 |
|------|---------|----------|
| **主标题/章节名** | 微软雅黑 (Microsoft YaHei) | 加粗 (Bold) |
| **封面信息 (标签/值)** | 楷体 (Kaiti) | 标签色 `#1E3A5F`，值色 `#000000` |
| **正文列表/正文编号** | Times New Roman / 微软雅黑 | 英文与标点优先 Times New Roman |
| **公式标注** | Times New Roman | 斜体 (Italic) |

## 4. 视觉层级
- 页面饱满度需 > 70%，禁止出现大面积空白。
- 字号根据内容多少在 22pt - 36pt 之间动态适配。
- 图像与文字间距需保持在 0.2 英寸以上。
